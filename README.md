A full-stack food delivery application built using the **Design Thinking approach**.  
This project connects **customers 🧑‍🍳**, **restaurants 🍽️**, and **delivery riders 🚴‍♂️** on one platform with **real-time order flow** and **live rider tracking 📍**.

---

## ✨ Features

### 🧑‍🍳 Customer Module
- 🔐 User registration and login  
- 🍔 Browse restaurants  
- 📋 View menu items  
- 🛒 Add items to cart  
- 📍 Add delivery address  
- 📦 Place orders  
- 💳 Payment integration support  
- 🗺️ Live rider tracking  

---

### 🍽️ Restaurant Module
- 🏪 Restaurant registration  
- 📍 Add restaurant location  
- 🍱 Manage menu items  
- 📦 Accept & prepare orders  
- 🔄 Update order status  

---

### 🚴 Rider Module
- 👤 Rider profile creation  
- 🟢 Availability toggle  
- 📦 Accept delivery orders  
- 🔄 Update delivery status  
- 📍 Share live location  

---

### ⚡ Real-Time Features
- 🔔 Real-time order updates  
- 📡 Rider assignment notifications  
- 📍 Live rider tracking  
- ⚙️ Socket-based updates
